# Skills

Agent skills curated by [Sanjeed](https://x.com/sanjeed_i). These extend coding agents with specialized capabilities.

## Available Skills

| Skill | Description |
|-------|-------------|
| [x-api-reader](x-api-reader/SKILL.md) | Read posts, threads, user timelines, and search X (Twitter) via the API v2 |

## Install

```bash
npx skills add sanjeed5/skills
```

Install a specific skill:

```bash
npx skills add sanjeed5/skills --skill x-api-reader
```

Install globally (available across all projects):

```bash
npx skills add sanjeed5/skills -g
```

Works with Cursor, Claude Code, Codex, Copilot, Cline, Windsurf, and [37 more agents](https://github.com/vercel-labs/skills#supported-agents).

## Browse Before Installing

```bash
npx skills add sanjeed5/skills --list
```
